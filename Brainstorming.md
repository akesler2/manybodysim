Brainstorming:

Project Goals:
- create simple simulations of many body systems using numerical methods to solve.
- practice using object oriented programming to create a modular system that can be expanded later.
- implement numerical methods to solve equations fo motion ie leapfrog, verlet, and runge kutta.
- Include force fields ie constant force, gravitational, electromagnetic, etc.
- include objects collision detection for walls and obstacles.
- implement emmiter objects that can emit particles or objects into the simulation.
- implement a simple GUI to visualize the simulations.

Requirements:
- Python 3.10 or higher
- numpy
- matplotlib
- streamlit?

Phase 1:
- create basic objects for universe, particles, etc. 
- initialize particles with random positions and velocities
- update state based on time step
- layout system architecture
- create a simple ui




Further goals:
